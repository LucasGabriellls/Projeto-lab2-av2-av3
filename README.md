# Super Kernel

## Bem-vindo ao SUPER KERNEL! Este é um projeto de sistema 
## de gerenciamento para um mercado, desenvolvido em Python.

# Funcionalidades
## Cadastro de Produtos: Adicione, edite e remova produtos do estoque.
## Consulta de Estoque: Verifique a disponibilidade e detalhes dos produtos.
## Gestão de Vendas: Registre vendas e calcule o total da compra.
